## 环境安装
```bash
npm install typescript --save-dev
npm install --save-dev ts-node-dev
```

```shell
npm install express
npm install --save-dev eslint @types/express @typescript-eslint/eslint-plugin @typescript-eslint/parser
```

- 初始化 tsconfig.json
```bash
 npm run tsc -- --init
```
